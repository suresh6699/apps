Icon Placeholder
================

For production builds, you need to add proper icon files:

- icon.ico (Windows) - 256x256 pixels
- icon.icns (macOS) - 512x512 pixels  
- icon.png (Linux) - 512x512 pixels

You can create these from a PNG image using online tools:
- https://icoconvert.com/ (for .ico)
- https://cloudconvert.com/ (for .icns)

For now, the app will use the default Electron icon.
